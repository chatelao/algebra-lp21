# Übungen: Gleichungen lösen

Löse nach $x$ auf und mache bei den ersten drei Aufgaben die Probe.

1. $x + 15 = 40$
2. $x - 7 = 12$
3. $4x = 32$
4. $x : 5 = 6$
5. $2x + 8 = 20$
6. $5x - 10 = 40$
7. $3x + 4 = x + 12$
8. $10 - 2x = 4$
9. $4(x + 2) = 20$
10. $\frac{x}{2} + 5 = 11$
11. $7x - 3 = 4x + 15$
12. $2(3x - 1) = 10$
