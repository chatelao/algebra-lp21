# Übungen: Gemischte Aufgaben

Hier kommt alles zusammen!

1. Vereinfache: $4(x+2) - 3x$
2. Löse: $2(x-3) = 14$
3. Einsetzen: Berechne $3x^2 - 5$ für $x=2$
4. Vereinfache: $-a + 4b - 3a - b$
5. Löse: $5x + 7 = 3x - 1$
6. Schreibe als Term: "Das Vierfache einer Zahl um 2 vermindert."
7. Multipliziere aus: $-x(x - 4)$
8. Löse: $12 - x = 20$
9. Vereinfache: $0.2x + 0.5x + 0.3y - 0.1y$
10. Löse: $\frac{2x}{3} = 4$
11. Klammere aus: $7ab + 14a$
12. Was ist der Unterschied zwischen $2x$ und $x^2$? Erkläre mit einem Beispiel für $x=3$.
