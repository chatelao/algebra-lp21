# Übungen: Distributivgesetz

Löse die Klammern auf oder klammere gemeinsame Faktoren aus.

### Ausmultiplizieren
1. $3(x + 4)$
2. $5(2a - 3)$
3. $x(y + 2)$
4. $4(3x - 5y + 2)$
5. $0.5(8b + 10)$
6. $-(x + 5)$
7. $2x(x + 3)$
8. $3(a + b) + 2(a - b)$

### Ausklammern (Faktorisieren)
9. $4x + 12$
10. $5a - 5b$
11. $10x + 15y$
12. $x^2 + 3x$
