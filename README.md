# Algebra-Begleiter

Ein KI-gestützter Mathe-Tutor für Schülerinnen und Schüler der 7. Klasse (Zyklus 3) in der Schweiz, ausgerichtet auf den **Lehrplan 21**.

Dieses Projekt bietet Lernmaterialien, Übungen und interaktive Unterstützung für Algebra-Themen wie Variablen, Terme, Gleichungen und mehr.

## Dokumentation

Die vollständige Dokumentation und alle Lektionen findest du online:

- 📖 **[Online-Dokumentation auf Read the Docs](https://algebra-lp21.readthedocs.io/)**
- 📄 **[PDF-Version herunterladen](https://algebra-lp21.readthedocs.io/_/downloads/pdf/latest/)**

## Features

- **Lehrplan 21 konform:** Speziell für den Zyklus 3 in der Schweiz entwickelt.
- **Sokratische Methode:** Der Tutor gibt nicht einfach Lösungen, sondern führt Schüler durch gezielte Fragen zum Ziel.
- **Umfangreiche Übungen:** Über 100 Aufgaben pro Thema mit Lösungen.
- **PDF-Export:** Alle Inhalte sind auch offline als PDF verfügbar.
